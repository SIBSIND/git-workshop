Hablemos de remotes

Un remote permite a varias personas colaborar en un mismo repositorio

git remote -v

git remote add origin https://github.com/foo/bar.git

git remote remove origin

Un repositorio puede tener tantos remotes como queramos

git remote add foo https://github.com/foo/bar.git

git remote add bar https://github.com/foo/bar.git

git remote add baz https://github.com/foo/bar.git
